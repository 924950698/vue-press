# Angular 项目目录

1. 项目

```
项目       
│
└─── 根模块 
│   
└─── 核心模块
|    └── 服务模块
|    └── Reducer 模块
|    └── Effects 模块
|
└─── 共享模块
     | —— 登录模块
     |    └──登录 / 注册
     |
     | —— 项目模块（懒加载）
          └── 项目列表 / 项目卡片 / 增 改 删 操作 / 邀请成员等
     |   
     | —— 任务模块（懒加载）
     |    └── 任务主视图 / 任务分组列表 / 任务列表 / 任务条目等
     |
     | —— 日历模块（懒加载）
          └── 日历视图
```
